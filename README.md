# Notes

To make Typescript work, need to set:

```json

{
    "workbench.iconTheme": "theia-file-icons",
    "typescript.tsserver.log": "normal",
    "typescript.tsdk": "./node_modules/typescript/lib"
}

```

in `settings.json`.
